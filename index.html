Check-in online obbligatorio

Compilate il check-in online prima dell'arrivo.

Codice di conferma:
Inserite le ultime 4 cifre del vostro numero di telefono seguite dalla data di check-in (GGMMYY).

Esempio:
telefono 3335984119 - 4119
check-in 05/04/2026 - 050426
codice - 4119050426

Usare solo numeri, senza spazi.

Online check-in required

Please complete the online check-in before arrival.

Confirmation code:
Enter the last 4 digits of your phone number followed by your check-in date (DDMMYY).

Example:
phone number 3335984119 - 4119
check-in date 05/04/2026 - 050426
code - 4119050426

Use numbers only, no spaces.

Check-in en ligne obligatoire

Veuillez compléter le check-in en ligne avant votre arrivée.

Code de confirmation :
Saisissez les 4 derniers chiffres de votre numéro de téléphone suivis de votre date d'arrivée (JJMMAA).

Exemple :
numéro 3335984119 - 4119
date d'arrivée 05/04/2026 - 050426
code - 4119050426

Utilisez uniquement des chiffres, sans espaces.
Check-in dalle 16:30 - Check-out entro le 10:30

Check-in à partir de 16h30 - Check-out avant 10h30
